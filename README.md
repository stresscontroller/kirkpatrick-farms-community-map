# Kirkpatrick Farms Community Map

Interactive community web map for Kirkpatrick Farms (Aldie, VA), built with Next.js + Leaflet.

This project provides a mobile/desktop-friendly map with:
- community amenities (ponds, pools, courts, clubhouse, etc.)
- trail lines
- category filters and search
- map style toggle (Satellite / Street)
- printable/shareable map view
- farm-boundary-restricted rendering

---

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Leaflet
- Radix UI / shadcn-style components

---

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### 3) Production build

```bash
npm run build
npm run start
```

---

## Available Scripts

- `npm run dev` - start local dev server (webpack mode)
- `npm run build` - create production build
- `npm run start` - run production server
- `npm run lint` - run ESLint
